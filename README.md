# Focus Tracker

This is a small project I built to track daily focus sessions and improve productivity.

I made this using Flask and SQLite to understand how backend, database and templates work together.

## What this project does

- Lets you add focus sessions
- Stores the data in a database
- Displays saved sessions
- Simple UI using HTML templates

## Why I built this

I wanted to practice:
- Flask basics
- Working with SQLite
- Handling forms
- Connecting frontend with backend

This project helped me understand how a real web app works.

## Tech Used

- Python
- Flask
- SQLite
- HTML

## How to Run

1. Clone the repository  
2. Create virtual environment  
   python -m venv venv  
3. Activate it  
4. Install Flask  
   pip install flask  
5. Run  
   python app.py  

Then open:  
http://127.0.0.1:5000
